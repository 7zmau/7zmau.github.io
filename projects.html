Moverr

Moverr is an inventory manager (kind of). With Moverr, you can keep track of items in your inventory. You can assign several "locations" in which you can store these items. It keeps track of the movements of all items happening in between locations. Try it out!

Loanapp

Loanapp is a loan management system API. It has several endpoints through which you can create a user, setup his/her account, get interest rates on the tenure selected and apply for a loan.
